Automatic Actions for LearnDash: [Actions] Enroll users to courses, add them to groups or change their roles on [Triggers] registration/login or on course complete. More actions and triggers to come!

== Description ==
Define an action (enroll user to courses, add user to group, change user role), connect it to a trigger (registration, login, course complete), sit back and relax. New actions and triggers will be available soon.

== Installation ==
1. Upload plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Click on the "LDActions" link (on the sidebar)
4. Create as many action-trigger pairs as you like.
